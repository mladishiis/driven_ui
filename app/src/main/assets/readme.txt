Структура архива с микроаппом

Корневая папка: содержит xml файл с микроаппом, который всегда называется microapp (содержит описание микроаппа и перечень экранов), папки mocks, queries, resources, screens
Папка mocks: содержит json файлы с моками для запросов screenQueries
Папка queries: содержит xml файл с рееестром запросов allQueries
Папка resources: содержит xml файл с реестром стилей allStyles и, опционально, файлы с контентом (например, изображения/иконки). Файлы с контентом могут быть только для дев-версии
Папка screens: содержит xml файлы с описанием отдельных экранов screen (для каждого экрана внутри описывается раздел с screenQueries), папку prompts
Папка prompts: содержит txt файлы с описанием промптов для каждого конкретного экрана, с тем же названием (названиеЭкрана.txt), если он создавался через генератор экранов.
